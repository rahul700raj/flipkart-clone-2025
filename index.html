To run the full application locally, follow these steps:

Frontend Setup

git clone https://github.com/rahul700raj/flipkart-clone-2025.git
cd flipkart-clone-2025/frontend
npm install
npm run dev

Access at: http://localhost:5173

Backend Setup

cd backend
npm install
cp .env.example .env
# Edit .env file
npm start

API runs on: http://localhost:5000
